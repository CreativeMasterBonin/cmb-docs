Thingamajigs

A decoration mod for MC

Welcome to Thingamajigs!

Thingamajigs adds furniture, appliances, road devices, railroad devices, seasonal features and more to your MC worlds!

The mod has been around since 2020. In the early days, the mod used cheap mod makers. In modern Thingamajigs, all code is from scratch (and open source components are used and credited when needed). A freak code and file corruption prompted the change from mod makers to made-from-scratch pure Java. An IDE was setup, some tips and tricks later, and boom, Thingamajigs' source code could now be on GitHub, releases on CurseForge and Modrinth, and not using any mod makers to boot. Code was improved as a result.

Nowadays, Thingamajigs has two versions:

  • Thingamajigs 1 (For Forge)
  • Thingamajigs 2 (For NeoForge)

Although Thingamajigs is under one name, all versions are different in some way. But not personalized, we don't do that here.

Thingamajigs 1 contains legacy and all content, no matter what category it is in. So balloons, statues, couches, entities, anything.

Thingamajigs 2 DOES NOT have all these features. It was split into 3 parts:

  • Thingamajigs (Base Mod)
  • Thingamajigs Long Roads
  • Thingamajigs Railroadways

A fourth mod called Thingamajigs Goodies had the balloons, purifying content, and other magic/tech stuff. It however did not feel like the right direction, so instead that content was scrapped in favor of another mod: Overpowered Mastery.
