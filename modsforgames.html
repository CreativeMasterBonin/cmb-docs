Mods For Games

CMB makes mods for many games, including MC, Cosmic Reach, and even some lesser known ones.

The cycle for Thingamajigs (and other mods like it) is as follows:

  • Add new content to either Thingamajigs 1/2.
  • Check content meets a standard of 'working' and 'polished enough'
  • Release new version with dedicated server tests
  • Add content from released version to version without the features
  • Check content meets a standard of 'working' and 'polished enough'
  • Release version with dedicated server tests
  • Update addons to match main core mod version
  • Check addon content meets a standard of 'working' and 'polished enough'
  • Release addon update version with dedicated server tests
  • Repeat cycle until mod is updated a few times and up to date with MC
